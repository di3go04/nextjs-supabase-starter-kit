# Next.js + Supabase SaaS Boilerplate — Source Code Acquisition

## What you're acquiring

You're buying the **full source code** and **commercial rights** to a production-ready Next.js 16 + Supabase + Stripe + Resend SaaS starter kit. The kit is already built, tested, and publicly listed on Gumroad at $50/copy. You can sell unlimited copies yourself.

This is NOT a website with traffic. This is a **digital product asset** — you acquire the code, you become the seller.

## Why buy this instead of building from scratch

Building a comparable SaaS boilerplate takes 3-6 months of senior dev time (~$15,000+ in salary). You get the same result for $499 in 1 day.

## What's included (22 features)

### 🔐 Auth & Security
- Magic Link + OAuth (Google/GitHub) via Supabase SSR
- RBAC middleware (user/free/premium/admin)
- Security headers (CSP, HSTS, X-Frame-Options)
- Rate limiting on auth endpoints (Upstash Redis)
- Zod validation on all server actions

### 💳 Billing
- Stripe Checkout + Billing Portal integrated
- **Idempotent Stripe webhook** (no double emails on retries)
- Dunning flow (handles `invoice.payment_failed`)
- Audit logs for admin actions

### 📊 Premium features (rare in $50 boilerplates)
- **Analytics dashboard** with Recharts (MRR, churn, growth)
- **In-app notifications** system (bell + dropdown + DB table)
- **API Keys management** + public `/api/v1/me` endpoint
- **Teams/Organizations** multi-seat with invitations
- **Admin panel** with MRR/churn metrics + inline role change

### 🌍 i18n & Emails
- ES/EN/PT actually wired (not scaffolding)
- 8 React Email templates + Resend integration
- Emails localized per user's locale

### 🧪 Quality & DevOps
- 38 tests passing (Vitest unit + Playwright e2e)
- Dockerfile multi-stage + docker-compose
- GitHub Actions CI (lint + typecheck + tests + build)
